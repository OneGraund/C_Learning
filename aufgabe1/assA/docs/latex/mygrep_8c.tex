\doxysubsection{Detailed Description}
This file implements a custom grep utility for searching keywords in files/stdin. 

This utility makes a line-\/by-\/line search for a specified by user keyword either case-\/sensitive or insesitive in multiple/signle files or from stdin stream

@synopsis mygrep \mbox{[}-\/i\mbox{]} \mbox{[}-\/o outfile\mbox{]} keyword \mbox{[}file...\mbox{]}


\begin{DoxyParams}{Parameters}
{\em -\/i} & Perform a case-\/insensitive search. \\
\hline
{\em -\/o} & Specify an output file to save search results \\
\hline
{\em keyword} & The keyword to search for in each line of the files/stdin \\
\hline
{\em file} & One or more files to search. If omitted, reads from stdin stream\\
\hline
\end{DoxyParams}
